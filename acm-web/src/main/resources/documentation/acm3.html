/plugin/casefile/file post

Manage case files.

post

Add a file to the case. Submitted form must include a caseId element, and a files[] element with the files to be uploaded. Returns JSON suitable for consumption by the jQuery File Upload plugin.

post /plugin/casefile/file

Add a file to the case. Submitted form must include a caseId element, and a files[] element with the files to be uploaded. Returns JSON suitable for consumption by the jQuery File Upload plugin.

  • Request
  • Response

Body

Type: multipart/form-data

Example:

# send a standard multipart form request.

HTTP status code 200

File(s) were successfully added.

Body

Type: application/json

Example:

{
          // NOTE: application/json is sent to all browsers EXCEPT Internet Explorer
          "files":[
            {
              "name":"filename.txt",
              "size":500,
              "url":"/acm/link/to/download/filename.txt",
              "thumbnailUrl":null,
              "deleteUrl":"/acm/link/to/delete/filename.txt",
              "deleteType":"DELETE",
              "error":null                        
            }
          ]                    
        }

Type: text/plain

Example:

{
          // NOTE: text/plain is sent ONLY to Internet Explorer.  Same format as application/json.
          "files":[
            {
              "name":"filename.txt",
              "size":500,
              "url":"/acm/link/to/download/filename.txt",
