/api/v1/plugin/ecm

/api/v1/plugin/ecm/download/byId/{ecmFileId} get

get

Retrieve the active version of the requested file; for example, if "2.0" is the active version, then version 2.0 is retrieved.

get /api/v1/plugin/ecm/download/byId/{ecmFileId}

Retrieve the active version of the requested file; for example, if "2.0" is the active version, then version 2.0 is retrieved.

  • Request
  • Response

URI Parameters

  • ecmFileId: required (integer)

    Object ID of the file to be downloaded.

Query Parameters

  • inline (boolean)

    Whether the file should be shown in the browser (true) or downloaded (false)

HTTP status code 200

Response body is the file content stream. The MIME type is appropriate for the actual file type, and could be any MIME type. If 'inline' was false, the Content-Disposition header is set to "attachment; filename=''" where is the actual file name.

HTTP status code 500

File could not be downloaded; either such a file does not exist, or the user does not have access to it.

Body

Type: text/plain

Example:

The response body will include an error message.

/api/v1/service/ecm/folder/copy/{folderId}/{dstFolderId}/{targetObjectType}/{targetObjectId} post

Copy a folder. API doc last verified on 2015-05-11.

post

Copy a folder to another location, either in the same container or a different container. The folder contents are recursively copied to the other location; this recursive copy operation will take time proportional to the number and size of objects to be copied.

post /api/v1/service/ecm/folder/copy/{folderId}/{dstFolderId}/{targetObjectType}/{targetObjectId}

Copy a folder to another location, either in the same container or a different container. The folder contents are recursively copied to the other location; this recursive copy operation will take time proportional to the number and size of objects to be copied.

  • Request
  • Response

URI Parameters

  • folderId: required (integer)

    Id of the folder to be copied. This folder must not be a top-level root folder; in other words, this folder's parentFolderId property must not be null.

  • dstFolderId: required (integer)

    Id of the target folder

  • targetObjectType: required (string)

    Object type of the object that will hold the copied folder

  • targetObjectId: required (integer)

    Object ID of the object that will hold the copied folder; the dstFolderId must be a folder owned by this object.

HTTP status code 200

Folder copied successfully.
