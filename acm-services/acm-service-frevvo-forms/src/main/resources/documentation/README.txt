DEPLOY STEPS (Frevvo v6.1):

1. Be sure that Frevvo server is stopped.
2. Create folder named "arkcase" under frevvo.war root
3. Extract "libs.zip" and add extracted folder under "arkcase"
4. Extract "rich-textarea-plugin-v3.0.zip" and add extracted folder under "arkcase"
5. Extract "user-picker-plugin-v1.0.zip" and add extracted folder under "arkcase"
6. Extract "patch.zip" and add extracted folder under "arkcase"
7. Start Frevvo server
8. Upload "ACM_ArmediaCaseManagement_app.zip" application (make sure that "Copy" and "Replace" checkboxes are selected)
