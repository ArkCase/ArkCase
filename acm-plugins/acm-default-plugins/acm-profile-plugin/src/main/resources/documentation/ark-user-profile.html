/api/v1/plugin/profile/outlook post

Update the user's Outlook password. The Outlook password is not a property of the UserOrg object, to prevent the password from being passed around with the UserOrg. With this approach the only time the password is passed around is when the user updates it.

post

Update the Outlook password for the authenticated user.
