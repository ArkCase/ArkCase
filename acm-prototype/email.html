  • When a user clicks "Send Email", it prompts the user to enter an email address or search for a username (same field)
  • If a user types in an email address and the email is already in the system, ArkCase will "autocorrect" the search result to the user's profile and allow them to add it to the "Recipients List" from the search results.
  • If an email address does not exist, it will allow the user to add the email address to the "Recipients List" from the search results.
  • The user can click "Add Selected Users" to add the selection to the "Recipients List" without sending an email, this way they can conitnue to search for other users before they're ready to send it.
  • If the users clicks "View Recipients List", it will lists the users's selection in the table.
  • When the user clicks "Send Email", the documents are then sent to the users in the "Recipients List".
